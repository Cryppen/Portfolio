An Arduino Programming Language for Primary School Students

Abstract

The purpose of this project was to design a new programming language suitable for teaching 7-9th grade students to learn programming through physical visualisation. The visualisation used for the programming language is an Arduino car constructed of LEGO. The designed language has statements for driving and turning the car. A syntactical specification is given using a CFG and a semantic specification is given using structural operational semantic. For the programming language, a working compiler is implemented in Java using ANTLR for constructing the lexer and parser part of the compiler. The compiler translates to C code. A usability test was conducted to evaluate the ease of the new programming language compared to the existing language for Arduino. The language is found to be more user friendly compared to the existing Arduino language.

My Contributions

Since the project is a 6 person collaboration I feel it's fair to outline my contributions to the project. The major aspect of the project I was in charge of was the implementation of the semantic and type checking constraints of the language. This involved implementating a symmbol table and its associated visitor. With the implementation of semantic and type checking rules good and understandable error messages were also some of my focus points.
